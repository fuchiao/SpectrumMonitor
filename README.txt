Spectrum Monitor

Description:
Showing real-time spectrum data received by serial port(RS232)
Recording data in csv
Loading and reviewing data from csv

Lib and Environment:
Python(x,y)2.7.5.1
pyserial-2.7

Usage:
Run this script, and input the size of the sample
Received number means verified data number
Wasted number means useless data number
Press RECORD button, it would start to save data with csv format.
Name of record file would be as same as the file path box.
If the file path box is empty, it would use the default filename, "tmp.csv".
Press LOAD button, it would load the file, which is indicated by the file path box
Press STOP button, you can finish every kind of action, and resume to the original state
